# FullStack Mini App — Frontend
 
Учебный fullstack-проект. Клиентская часть приложения с авторизацией, регистрацией и загрузкой файлов.
 
---
 
**Репозиторий бэкенда:** https://github.com/KettyPe/FullStack-Mini-App-Backend
 
---
 
## О проекте
 
Первый опыт работы с fullstack-архитектурой.  
Фронтенд взаимодействует с собственным REST API на Node.js + Express.
 
### Что реализовано
 
- Регистрация и авторизация пользователя
- JWT-аутентификация — хранение токена в LocalStorage
- Загрузка файлов на сервер
- Взаимодействие с бэкендом через Axios
- Защищённые маршруты для авторизованных пользователей
- Текстовый редактор на базе Simple Editor
 
---
 
## Стек
 
- JavaScript, HTML, SCSS
- React, React Router
- Redux Toolkit — стейт-менеджмент
- Axios — HTTP-запросы к API
- JWT — аутентификация
- LocalStorage — хранение токена
- Simple Editor — текстовый редактор
 
---
 
## Запуск локально
 
```bash
npm install
npm start
```
 
> Для работы приложения необходимо запустить бэкенд — см. репозиторий бэкенда.
